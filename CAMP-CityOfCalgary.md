## Research Notes

### How the liveandplay.calgary.ca site works
- **JavaScript-rendered** — must use Playwright; plain fetch/curl returns no camp data.
- Each camp type has a direct URL using `CourseTypes=<GUID>` and `SortResultsMode=CourseStartDate`.
- Pages paginate; look for `link "2"` / `link "3"` refs near end of snapshot, then click them.
- Each session block contains: heading (with age group in parentheses), Price, Spaces, From date, To date, then a schedule table with days/hours/location and a Google Maps link.
- **Spaces: 0 = sold out.**
- The 4-day August week (Aug 4–7) is priced ~75% of regular because Mon Aug 3 is Heritage Day.

### Parse script
A reusable Python parse script lives at:
`research/city-of-calgary/parse.py`

All downloaded page snapshots (JSON) are in:
`research/city-of-calgary/`

Run: `python3 parse.py arts-ventures-p*.json` etc. to re-parse any camp.
